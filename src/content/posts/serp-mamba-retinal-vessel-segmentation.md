# 论文笔记
### 论文速读
- **不懂的词**：
	- 血管拓扑一致性的先验知识[21]
	- 保证分段血管得到有效关注并保持相干性
	- 指标 Dice、IoU 和 MCC 的含义都是什么，如何进行计算的？
- **Abstract**
	- **任务**：高分辨率血管分割
	- **数据集**：UWF-SLO
	- 捕捉视网膜的高分辨率视图
	- 展度为200度
- **Introduction**
	- **难点**：
		1. 血管形状本身就很复杂
		2. **高分辨率导致的难点**：
			1. 高分辨率加剧目标和背景的类别不平衡
			2. 眼睑和睫毛覆盖了血管
			3. 不均匀的照明和对比度
	- **SSM**：长程依赖关系建模（复杂度：线性级别）
		- 但是无法适应血管的弯曲和变化，无法维持血管的连续性
	- **创新点**：
		- **创新点1**（SIA）：改变Mamba块的扫描路径——蛇形扫描路径
			- 这使得Mamba能够克服方向敏感性问题，同时确保高效提取高分辨率图像中的连续血管特征。
		- **创新点2**（ADDR）：模糊驱动双重校准（ADDR）模块——缓解类别失衡
			- 开发了连续性感知机制，用于检测模糊像素周围的像素值，从而确保血管的连续性。
			- 通过血管和背景的双重驱动来细化模糊像素
	- **贡献点**：
		1. 使用SSM
		2. SIA
		3. ADDR
		4. **结果**：在三个UWF-SLO数据集上的实验表明，我们的方法优于其他容器、超分辨率和基于曼巴的分割方法。
- **Method overview figure**

- **Conclusion**
	- **论文核心贡献**：
	    - 提出了 **Serp-Mamba**，一种基于 Mamba 的方法，专门用于高分辨率 **UWF-SLO** 图像的视网膜血管分割。
	    - 主要创新点是引入了 **SIA（Serpentine Interwoven Adaptive）** 蛇形自适应扫描机制和 **ADDR（Ambiguity-Driven Dual Recalibration）** 模块，分别解决了血管的连续性问题和类别不平衡问题。
	- **实验结论**：
	    - **Serp-Mamba** 在多个数据集（例如 **PRIME-FP20** 和 **MU-VS** 数据集）上表现优越，取得了较高的 **Dice**、**IoU** 和 **MCC** 分数，超过了其他基线方法（如 U-Mamba、SCS-Net 等）。
	    - 通过消融实验，验证了 **SIA** 和 **ADDR** 两个模块的独立性和有效性，证明它们能显著提高分割效果。
	- **未来工作方向**：
	    - 在 **SIA** 模块的基础上进一步优化血管扫描路径，特别是调整路径的宽度以更适应不同血管的直径。
	    - 增强 **ADDR** 模块，进一步提高对模糊区域的处理能力，尤其是在低对比度图像中。
	- **笔记总结**：
		- 这篇论文提出的 **Serp-Mamba** 方法，针对高分辨率视网膜图像中的血管分割问题，做了结构性创新。通过 **SIA** 和 **ADDR** 模块的结合，解决了血管连续性和类别不平衡的问题，实验结果表明该方法在多个数据集上表现优于现有方法。未来的工作可以进一步优化扫描路径和模糊区域的分类。
- **Ablation tables**
	- **实验设计**：
	    - 消融实验通过逐个去除创新模块，验证 **SIA** 和 **ADDR** 模块对性能的贡献。
	    - 不同配置实验：
	        - **Baseline**：只使用传统的 **U-Mamba** 模型。
	        - **M1**：在 **U-Mamba** 基础上添加 **SIA** 模块。
			- **M2**：在 **U-Mamba** 基础上添加 **ADDR** 模块。
	        - **Ours**：结合 **SIA** 和 **ADDR** 的 **Serp-Mamba** 模型。
	- **实验结果**：
	    - **SIA** 提高了 Dice、IoU 和 MCC，证明蛇形扫描有效增强了血管的提取。
	    - **ADDR** 模块有效处理了类别不平衡问题，并且改善了模糊区域的分类，提高了模型的鲁棒性。
	    - 最终的 **Serp-Mamba** 模型，结合了 **SIA** 和 **ADDR**，达到了最佳性能。
	- **结论**：
	    - **SIA** 和 **ADDR** 模块不仅各自提高了模型的性能，而且两者结合后的结果显示出**显著的性能提升**。
	- **笔记总结**：
		- 通过消融实验，验证了 **SIA** 和 **ADDR** 模块的有效性，证明这两个模块分别在血管连续性和类别不平衡问题上起到了关键作用，最终的 **Serp-Mamba** 模型集成了这两部分的优势，达到了更高的分割精度
- **Result tables**
	- **定量结果**：
	    - 在 **PRIME-FP20** 数据集上，**Serp-Mamba** 达到了 **69.01%** 的 **Dice**，**52.93%** 的 **IoU**，**68.41%** 的 **MCC** 和 **65.75%** 的 **BM**，超越了其他传统方法（例如 **U-Mamba**、**SCS-Net**）。
	    - 在 **MU-VS Center A** 和 **MU-VS Center B** 数据集上，**Serp-Mamba** 同样展示了较好的性能，尽管 Center B 数据集的复杂性更高，模型依然稳定地优于其他方法。
	- **定性结果**：
	    - 论文中展示了多个定性图像，比较了 **Serp-Mamba** 和其他方法（如 **U-Mamba**）的分割效果。**Serp-Mamba** 在分割血管细节、处理低对比度和遮挡区域方面表现出色，显著减少了误分和漏分。
	- **对比其他方法**：
	    - 与 **U-Mamba** 等方法对比，**Serp-Mamba** 通过 **SIA** 和 **ADDR** 模块的创新，使其在血管分割精度、连通性和类别不平衡处理方面都取得了较大的提升。
	    - 该方法在处理复杂图像和不清晰血管时，表现出更强的鲁棒性。

- **对比**：以下为序列模型
	- **Transformer**
		- **核心**：self-attention
		- **优点**：
			- 长距离依赖关系
			- 平行计算
		- **缺点**：相当于一直在做大矩阵乘法
			- 计算很大
			- 推理速度慢
		- **表示**：
			- 对于一个序列，长度为 `N`，维度为 `d`，复杂度为 `O(N^2d)`
				- ![serp-mamba-20260424202056](/images/serp-mamba-20260424202056.png)
				- 对于第 `i` 行，要和 `N` 列相乘再相加
				- 单个维度，矩阵点乘的数量：`N*N`
				- 因为维度是 `d` 维度，矩阵点乘法的数量是：`N*N*d`
	- **RNN**：
		- **核心**：
			- 当前状态来自前一个状态+输入：`h_t=f(x_t,h_{t-1})`
			- 当前输出来当前状态：`y_t=f(h_t)`
		- **优点**：推理很快
		- **缺点**：
			- 会遗忘，无法建模长距离依赖关系
			- 只能串行计算，训练慢
	- **Mamba**：
		- **状态方程**：`h'(t)=Ah(t)+Bx(t)`
			- 当前状态的变化率来自前一个状态+当前输入
		- **输出方程**：`y(t)=Ch(t)`
			- `y(t)=Ch(t)+Dx(t)` 这里的第一条路径被忽略
				- ![serp-mamba-20260424203131](/images/serp-mamba-20260424203131.png)
	- **S4**：基于SSM改进
		1. **离散化SSM**：
			- 零阶保持技术
				- `A_bar = exp(Delta A)`
				- `B_bar = (Delta A)^(-1)(exp(Delta A)-I) * Delta B`
			- 变换：
				- `h_k = A_bar h_{k-1} + B_bar x_k`
				- `y_k = C h_k`
		2. **卷积结构表示**：
			- 不断展开状态 `h`
			- 提取系数 -> 矩阵乘法 -> 并行计算
				- ![serp-mamba-20260424204733](/images/serp-mamba-20260424204733.png)
		3. **解决长距离依赖关系**：HiPPO表示
			- 从类似RNN（会遗忘）-> 有较好的长距离建模
	- **Mamba（S6）**：
		1. **对输入信息有选择性处理**
			- 动态参数设计：让参数随输入的变化而变化 -> 有选择性地处理
				- ![serp-mamba-20260424205339](/images/serp-mamba-20260424205339.png)
		2. **硬件感知的设计**：并行扫描算法 -> 加快训练速度
		3. **更简单的SSM架构**：
			- 将H3和MLP进行综合：类似Transformer中的block
				- ![serp-mamba-20260424205520](/images/serp-mamba-20260424205520.png)
